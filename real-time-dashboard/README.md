# Real-Time Analytics Dashboard

Track and visualize live metrics using:

- Express.js backend
- MongoDB database
- Socket.IO for real-time data

## Folder Structure

- `models/` — Mongoose schema
- `config/` — Database connection
- `controllers/` — Functional logic
- `routes/` — API routes
- `public/` — Frontend (HTML)
